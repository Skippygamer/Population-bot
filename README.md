## Setup Instructions

### Linux

1. **Create a new directory**
    ```shell
    mkdir my-discord-bot
    cd my-discord-bot
    ```

2. **Initialize a new Node.js project**
    ```shell
    npm init -y
    ```

3. **Install dependencies**
    ```shell
    npm install discord.js node-fetch
    ```

4. **Start the bot**
    ```shell
    node index.js
    ```
